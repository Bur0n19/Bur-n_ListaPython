#  Fundamentos de Inteligencia Artificial - 1er Parcial - 6IV8

Este repositorio contiene la colección de ejercicios prácticos desarrollados para el primer parcial de la materia **Fundamentos de Inteligencia Artificial**. 

El proyecto consiste en una aplicación de escritorio interactiva, construida en Python, que resuelve diversos problemas de lógica de programación, validación de datos, control de flujo y procesamiento de información a través de una interfaz gráfica moderna.

#Tecnologías y Herramientas Utilizadas

* Lenguaje base: Python
* Interfaz Gráfica (GUI): CustomTkinter (Implementación de ventanas y frames mediante Programación Orientada a Objetos).
* Gestión de Datos: Pandas (Utilizado para la estructuración y visualización de historiales y reportes en formato tabular).
* Entorno de Desarrollo (IDE): PyCharm

#Contenido del Proyecto

El sistema cuenta con un menú principal que centraliza el acceso a los diferentes ejercicios solicitados para la evaluación. Entre los módulos destacados se encuentran:
* Manejo de Sueldos (Ej. 1) Calcula el aumento del sueldo del trabajador segú su sueldo base.
* Sistemas de Registro y Costos (Ej. 2 y 3): Cálculo de tarifas con descuentos dinámicos (por edad o mes) y generación de historiales de recaudo.
* Validadores Lógicos con Memoria (Ej. 4, 5 y 6): Algoritmos de verificación de rangos numéricos con contadores de intentos y registro de historial en tiempo real.
* Acumuladores Condicionados (Ej. 8 y 9): Sistemas de sumatorias con restricciones lógicas (ej. detención automática al superar el valor de 100) y limpieza de datos.
* Calculadora de Nómina (Ej. 10): Sistema integral para el cálculo de salarios de trabajadores, integrando horas normales, horas extras y bonificaciones familiares, exportado a un reporte visual.
